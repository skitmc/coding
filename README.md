# DevLabs

An interactive web-based platform for learning HTML and JavaScript through study guides, coding challenges, and quizzes.

## Features

- **Study Guides**: Comprehensive guides on HTML and JavaScript basics
- **Coding Challenges**: Interactive code editor with real-time feedback and hints
- **Quizzes**: Multiple-choice questions to test your knowledge
- **Error Highlighting**: See errors and corrections when running code
- **Score Tracking**: Keep track of your progress

## How to Use

1. Open `index.html` in a web browser
2. Navigate through the sections using the top menu
3. Start with Study Guides to learn concepts
4. Try Challenges to practice coding
5. Test your knowledge with Quizzes

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Getting Started

Simply open the `index.html` file in any modern web browser. No server required for basic functionality.
